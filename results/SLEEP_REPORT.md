# 아침 수면 리포트 (클라우드 · 비실시간)

> 엣지가 업로드한 **비식별 이벤트/특징만** 사용한다. 원시 음성·레이더 신호는 전송하지 않는다.

## 야간 요약 (8.0시간 모니터링)

| 항목 | 값 |
|---|---|
| 울음 이벤트 | 4회 (총 10분) |
| 무호흡 의심 이벤트 | 1회 |
| 평균 호흡수 | 39.9 회/분 (범위 32.7~45.7) |
| 최장 안정 수면 | 298분 연속 |
| CO₂ 범위 | 559~893 ppm |
| 실내 온도 범위 | 20.9~23.2 ℃ |

## 시간대별 울음 분포

| 시간대 | 울음 |
|---|---|
| 4시간대 | 2분 |
| 5시간대 | 3분 |
| 7시간대 | 5분 |

## 울음-맥락 상관

- 울음이 7시간대에 가장 잦음(5회)

## 개인화(추세 기반) 제언

- 학습된 개인 정상 호흡 범위: (31.7, 47.2) 회/분 (이 아기 기준으로 임계값 보정).
- 최장 안정 수면 298분 기준으로 가구별 baseline을 갱신한다.
- CO₂가 893ppm까지 상승 → 취침 전 환기 또는 공기청정 세기 상향을 제안.

## 한계

- 본 리포트는 합성 야간 데이터 기반이며, 실제 영유아·임상 검증이 아니다.
